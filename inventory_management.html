🔗 Google Sheets Setup Guide

This app connects to Google Sheets as a free live database via Google Apps Script. Setup takes ~5 minutes.

  1. Open Google Sheets and create a new spreadsheet named MoldPartsInventory.
  2. Create two tabs: Items and Categories.
  3. In Items tab, row 1 headers: id · name · category · revs
    The revs column stores JSON like [{"rev":"A","qty":3}]
  4. In Categories tab, row 1 header: name. Add category names below.
  5. In Google Sheets, click Extensions → Apps Script.
  6. Delete all existing code, paste the script below, then Save (Ctrl+S).
  7. Click Deploy → New Deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy & authorize.
  8. Copy the Web App URL and paste it into the SHEET_URL constant at the top of the HTML file's script section.
  9. Click ↻ Sync from Sheet to load your data.
