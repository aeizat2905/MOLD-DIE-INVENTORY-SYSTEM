🔗 Google Sheets Setup Guide

Connect to Google Sheets as a free live database via Apps Script (~5 min setup).

  1. Create a new Google Spreadsheet named MoldPartsInventory.
  2. Add two tabs: Items and Categories.
  3. Items tab row 1 headers: id · name · category · revs
  4. Categories tab row 1 header: name, then add category names below.
  5. Click Extensions → Apps Script, delete existing code, paste the script below.
  6. Click Deploy → New Deployment. Type: Web app. Execute as: Me. Access: Anyone. Click Deploy & authorize.
  7. Copy the Web App URL and replace YOUR_APPS_SCRIPT_URL_HERE in the HTML file's SHEET_URL constant.
  8. Click ↻ Sync from Sheet.
